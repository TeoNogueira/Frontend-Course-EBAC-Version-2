Hoje vemos como padrão para desenvolvimento web essa trilogia citadas como: HTML, CSS e Javascript.
 A importância de usar essas duas linguagens de marcação (HTML e CSS) e linguagem de programação javascript,
 facilita o trabalho dos desenvolvedores por causa do seu dinamismo com a linguagem de fácil acesso e grande comunidade mundial,
 para auxiliar dúvidas e trazer melhor resultado para o usuário final.
 
- Por mais que tenha sido uma jogada de marketing por causa do "JAVA" o javascript é praticamente íncrivel usando com a combinação HTML + CSS.
- Outra vantagem de se ter o javascript no html e css, é a quantidade de bibliotecas que existe para auxiliar um time de desenvolvedores.
